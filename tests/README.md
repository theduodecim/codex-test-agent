# Tests con Vitest

No se generaron tests unitarios porque este repositorio no contiene una carpeta `src/` ni archivos de código fuente para cubrir.

Cuando exista código en `src/`, se recomienda:
- instalar y configurar Vitest
- crear specs en `tests/` por cada módulo en `src/`
- incluir casos nominales, edge cases y validaciones de errores
